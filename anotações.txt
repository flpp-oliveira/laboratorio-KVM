CUSTOMIZE

# Este script utiliza o comando virt-customize, que é uma ferramenta para personalizar imagens de máquinas virtuais antes de serem iniciadas.

# Ele está realizando as seguintes ações:

#     -a focal-server-cloudimg-amd64.img: Especifica a imagem que será personalizada.

#     --timezone "America/Sao_Paulo": Configura o fuso horário da máquina virtual para America/São Paulo.

#     --update: Atualiza todos os pacotes instalados na imagem.

#     --network: Configura a rede da máquina virtual.

#     --uninstall cloud-init: Remove o pacote cloud-init da imagem.

#     --root-password password:85885072: Define a senha do usuário root para 85885072.

#     --firstboot-command "dpkg-reconfigure -f noninteractive openssh-server": Executa o comando dpkg-reconfigure -f noninteractive openssh-server na primeira inicialização da máquina virtual. Este comando reconfigura o pacote openssh-server.

#     --run-command 'sed -i "s/PasswordAuthentication no/PasswordAuthentication yes/" /etc/ssh/sshd_config': Executa o comando sed para alterar a configuração do openssh-server, permitindo autenticação por senha.

#     --run-command 'sed -i "s/#PermitRootLogin prohibit-password/PermitRootLogin yes/" /etc/ssh/sshd_config': Executa outro comando sed para permitir o login como root usando senha.
